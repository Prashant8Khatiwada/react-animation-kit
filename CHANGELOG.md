# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project setup
- Core entrance animations: `fadeIn`, `fadeInUp`, `slideUp`, `slideDown`, `slideLeft`, `slideRight`, `zoomIn`
- Core exit animations: `fadeOut`, `fadeOutUp`, `slideOutUp`, `zoomOut`
- Custom easing curves: `smooth`, `softOvershoot`, `bounce`, `snap`, `decelerate`, `accelerate`, `standard`
- Spring configuration presets: `gentle`, `bouncy`, `snappy`, `slow`
- Duration and delay presets
- `useReducedMotion` hook for accessibility
- `useInView` hook for scroll-triggered animations
- Full TypeScript support
- Tree-shakeable build configuration
- SSR-safe implementation
